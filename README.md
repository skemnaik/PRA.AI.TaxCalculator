UK employee income tax and National Insurance calculator — a React + Vite web app deployed on Vercel. Entire delivery from PRD, phased implementation plan, to production code driven by AI tooling.

Problem Statement
UK employees struggle to understand take-home pay. Existing calculators are cluttered with ads or lack clear breakdowns of Income Tax bands, NI contributions and net salary projections.

Key Features
Gross-to-net salary calculation covering all UK Income Tax bands (2024/25)
National Insurance (NI) contribution breakdown by Class 1 thresholds
Monthly and annual take-home summary with visual tax split
HMRC-compliant tax band logic with configurable tax year
Mobile-responsive layout — zero ads, clean UX
Vite HMR for fast iteration; ESLint-enforced code quality

Tech Stack
React 18, Vite, JavaScript, CSS Modules, ESLint, Vercel, Antigravity AI
